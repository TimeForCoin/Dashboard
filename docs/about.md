# 关于项目

**闲得一币**是大学生通过做任务挣钱的云平台，定位上是重点在于运营的众包系统。在本平台中，每个人可以是挣闲钱的受益者，也可以是所谓的“奶牛”。

平台提供两大类任务分类：

- 问卷类
  - 问卷调查
  - 简历收集
- 非问卷类
  - 信息类，比如寻找学习资料等
  - 跑腿类，比如快递、外卖、寻物启事等。

同时，平台还支持公益性的消息公示，比如失物招领。
